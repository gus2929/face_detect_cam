mini project <얼굴 위치 추적 카메라><br/>

물체가 화면 중심에 위치할 때까지 pan과 tilt를 조절합니다.<br/>

pan과 tilt의 조절은 서보 모터를 특정 각도로 이동시키는 것으로 이해됩니다.<br/>

pan과 tilt의 이동 방향과 정지 조건은 물체의 중심 위치에 따라 결정됩니다.<br/>
